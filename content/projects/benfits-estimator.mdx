---

- Estimates eligibility, length of time, and amount of money applicant qualifies for. 
- Developed using Next.js, NestJS, and Azure. 
- Owned development of several frontend pages and creation of RESTful API using Axios. 
- Worked with adjacent teams to address security vulnerabilities and integrate analytics.
- Deployed using Docker and Azure Pipelines, enhancing deployment efficiency and automating the CI/CD process.
- Provides estimates and eligibility information in 5-10 minutes, when previously users could wait up to 28 days for eligibility results.
